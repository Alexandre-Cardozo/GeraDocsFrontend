Form-section card (ETP editor); `CardPanel` (same file) is the header-row card used for tables and lists.

```jsx
<SectionBlock title="Estimativa das Quantidades" hint="Informe as quantidades estimadas com base no histórico de consumo...">
  ...fields...
</SectionBlock>
<CardPanel title="Processos Recentes" action={<a href="#">Ver todos →</a>}>...table...</CardPanel>
```
